# DesktopClean
文件整理软件

# 使用前请配置 ./resource/config/application.conf
# 如何设置新分类
```
图片:IMAGE_TYPE[
    .jpg,
    .jpeg,
    .png
]
```

# 如何添加新路径
```
PATH::下载整理[
    source:/Users/weigaolei/Downloads,
    out:/Users/weigaolei/Downloads,
    hasDir:true
]
```

source:你要整理的文件夹路径
out:将文件整理归档的文件夹
hasDir:是否将待整理文件夹下的所有子文件夹里的内容一同整理
